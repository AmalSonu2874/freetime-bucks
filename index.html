    Breadth first search is the most common search strategy for traversing a tree or a graph .This algorithm searches breadth wise in a tree or graph , So it is called breadth first search.

  • BFS algorithm starts searching from the root node of the tree and expands all success or node at the current level before moving to nodes of the next level.
  • The breadth -first search algorithm is an example of a general - graph search algorithm
  • Breadth-first search implemented using FIFO queue data structure.

    2.DEPTH FIRST SEARCH.


    Depth first search is a recursive algorithm. for traversing a tree or graph data Structure


  • It is called the depth-first search because it Starts from the root node and follows each path to its greatest depth node before. moving to the next path.
  • DFS uses a stack data structure for its implementation
  • The process of the DFS agoristim is similar to the BFs algorithm.

    • Note: Back tracking is an algorithm technique it for finding all possible solutions using recursion

    ADVANTAGES :

  • DFS requires less memory as it only needs to store a stack of the nodes on the path from root node to the current mode.
  • It takes less time to reach to the goal mode than BFS algorithm (if it traverses in the right path)
  • With the help of this we can stordes the route Which is being tracked in memory to save time as it only needs to keep one at a particular time.

    • DIS ADVANTAGES :

  • There is the possibility that Keep re-occuring, and there is no gurantee of finding the solution.
  • DFS algorithm goes for deep down searching and sometime it may go to the infinite loop.
  • The depth first search (DFS) algorithm does not always find the shortest path to a solution

    This can give you a brief summary about the talk by shashi taroor at oxford university.


  • 🇮🇳 Tharoor cites India’s economic decline from 23% to below 4% of global trade during British rule.
  • 💰 The British exploited India for resources, financing their Industrial Revolution at India’s expense.
  • 🏭 Colonial policies dismantled India’s textile industry, leaving weavers impoverished.
  • 📜 Reparations principles are likened to other historical precedents, such as Germany’s reparations post-WWII.
  • 🍽️ Starvation in India, exacerbated by British policies during famines, illustrates colonial neglect.
  • 🚂 British infrastructure was built primarily for colonial interests, not local benefits.
  • 🤝 The acknowledgment of historical wrongs is essential for reconciliation and reparations.

    Key Insights


  • ⚖️ Colonial Economic Impact: Tharoor illustrates how British colonialism drastically reduced India’s economic power, transforming it from a leading exporter to an impoverished nation. This underscores the long-lasting effects of exploitation.
  • 🏭 Deindustrialization: The dismantling of India’s textile industry highlights how colonial policies were designed to benefit Britain rather than the local economy, leading to widespread poverty among skilled artisans.
  • 📉 Moral Responsibility: Tharoor emphasizes that acknowledging historical injustices is a moral obligation, advocating for reparations not just as financial compensation but as recognition of past wrongs.
  • 🌍 Global Context: He draws parallels with other nations that have received reparations, reinforcing that such actions are not unprecedented and can promote healing and accountability.
  • 🥴 Human Cost of Colonialism: The reference to famines and starvation illustrates the human suffering caused by colonial policies, challenging the narrative of benevolent governance.
  • 🚂 Infrastructure Misuse: The argument that railways were built for British benefit rather than Indian needs reveals the exploitative nature of colonial investments, questioning the legacy often celebrated by former colonizers.
  • 💔 Long-term Consequences: Tharoor’s insights reflect on how colonialism has shaped contemporary issues in post-colonial societies, suggesting that reparations could aid in addressing these deep-rooted challenges.
