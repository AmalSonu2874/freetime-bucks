    • 2.Seach Algorithms

    SEARCHING : It is a step-by-step procedure to solve a search problem in agiven search space factors of search problems are :

  • SEARCH SPACE : It represents all the possible path which system may have.
  • START STATE : It is a state from where agent begins the search.
  • GOAL TEST : It is a function that observes the current state and return whether the goal state is achieved or not .

    • SOME BASIC KEYWORDS :

  • SEARCH TREE : Tree representation of search problems is called search tree . Thr root of search tree . The root of search tree is the root node which corresponding to the initial state.
  • ACTIONS : It gives the description of all the available actions to the agent.
  • PATH COST : It is a function that assign numeric cost to each path.
  • TRANSITION MODEL : A description of what each action do , can be represented to transiton model.
  • SOLUTION : It is an action sequence which leads fro the srart node to the goal node.
  • OPTIMAL SOLUTION : If a solution has the lowest cost among all solutions.

    • EVALUATION METRICS / PROPERTIES OF SEARCH.

  • Completeness - A search algorithms is said to be complete of it guarantees to return a solution if at least any solution exists for any random input.
  • Optimality - If a solution found for an algorithms is guaranteed to be the best solution (lowest path cost) among all other solutions then such as solution for is said to be optimal solution.
  • Time Complexity - Time complexity is a measure of time for an algorithms to complete its task.
  • Space Complexity - Maximum storage soace is required at any point during the search as complexity of problem .

    1. BREADTH FIRST SEARCH.


    Breadth first search is the most common search strategy for traversing a tree or a graph .This algorithm searches breadth wise in a tree or graph , So it is called breadth first search.

  • BFS algorithm starts searching from the root node of the tree and expands all success or node at the current level before moving to nodes of the next level.
  • The breadth -first search algorithm is an example of a general - graph search algorithm
  • Breadth-first search implemented using FIFO queue data structure.

    ADVANTAGES :

  • BFS will provide a solution if any solution exists
  • If there are more than one solutions for a given problem , then bfs wil provide the minimal solution which requires the least numbe rof steps.
  • It also helps in finding the shortest path in goal state , since it needs all nodes at the same hierarchical level before making a move to nodes at lower levels.

    1.‘Britain Does Owe Reparations’ by Shashi Tharoor

    Speech by Tharoor


    SUMMARY

    This can give you a brief summary about the talk by shashi taroor at oxford university.

  • Tharoor cites India’s economic decline from 23% to below 4% of global trade during British rule.
  • The British exploited India for resources, financing their Industrial Revolution at India’s expense.
  • Colonial policies dismantled India’s textile industry, leaving weavers impoverished.
  • Reparations principles are likened to other historical precedents, such as Germany’s reparations post-WWII.
  • Starvation in India, exacerbated by British policies during famines, illustrates colonial neglect.
  • British infrastructure was built primarily for colonial interests, not local benefits.
  • The acknowledgment of historical wrongs is essential for reconciliation and reparations.

    • KEY INSIGHTS

  • Colonial Economic Impact: Tharoor illustrates how British colonialism drastically reduced India’s economic power, transforming it from a leading exporter to an impoverished nation. This underscores the long-lasting effects of exploitation.
  • Deindustrialization: The dismantling of India’s textile industry highlights how colonial policies were designed to benefit Britain rather than the local economy, leading to widespread poverty among skilled artisans.
  • Moral Responsibility: Tharoor emphasizes that acknowledging historical injustices is a moral obligation, advocating for reparations not just as financial compensation but as recognition of past wrongs.
  • Global Context: He draws parallels with other nations that have received reparations, reinforcing that such actions are not unprecedented and can promote healing and accountability.
  • Human Cost of Colonialism: The reference to famines and starvation illustrates the human suffering caused by colonial policies, challenging the narrative of benevolent governance.
  • Infrastructure Misuse: The argument that railways were built for British benefit rather than Indian needs reveals the exploitative nature of colonial investments, questioning the legacy often celebrated by former colonizers.
  • Long-term Consequences: Tharoor’s insights reflect on how colonialism has shaped contemporary issues in post-colonial societies, suggesting that reparations could aid in addressing these deep-rooted challenges.

  • Global Collaboration: The proposed World Knowledge Platform illustrates the potential of joint efforts in innovation and problem-solving, reinforcing the idea that collaboration leads to mutual benefits.
